## Client Installation Guide
Follow these steps to install and start using Nightsoul Client.
### Step 1: Install Java

Minecraft requires Java to run properly.

1. Download the latest version of Java.
2. Install it on your computer.
3. Restart your PC after installation (recommended).

### Step 2: Install a Minecraft Launcher

You need a launcher to run Minecraft.

1. Install the official Minecraft Launcher or another supported launcher.
2. Open the launcher once to make sure Minecraft installs correctly.

### Step 3: Download Vertex Client

1. Go to the **#client-download** channel in our Discord server.
2. Download the latest **Client .jar file**.

### Step 4: Locate Your Minecraft Folder

1. Press **Windows + R** on your keyboard.
2. Type:

```
%appdata%
```

3. Open the **.minecraft** folder.

### Step 5: Install the Client

1. Find the **mods** folder inside `.minecraft`.
2. Drag and drop the **Client .jar file** into the **mods** folder.

*(If the mods folder does not exist, create a new folder and name it `mods`.)*

### Step 6: Launch Minecraft

1. Open your Minecraft launcher.
2. Select the version compatible with Client.
3. Click **Play**.

### Step 7: Verify Installation

Once Minecraft starts, Client should load automatically.
You should see the client interface or menu in-game.
### Need Help?

If you run into any issues during installation, open a ticket in the **#support** channel on our Discord server.
